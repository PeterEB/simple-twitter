# simple-twitter

The simple twitter.

## Quick Start

1. Install Docker and Docker Compose
2. `git clone https://github.com/PeterEB/simple-twitter.git`
3. `cd simple-twitter`
4. `docker-compose up --build`
5. Visit `http://127.0.0.1:3000/graphql` to test GraphQL APIs
